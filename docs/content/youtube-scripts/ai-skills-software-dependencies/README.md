Lead with `skills-are-dependencies`. It is the cleanest thesis match and should
still make sense after the gstack news cycle has moved on. Use
`eval-your-skills` if the final title leans harder into evals.

## Thumbnail prompt notes

The thumbnail should sell the broader engineering idea rather than only the
gstack repo. Use gstack as proof inside the video, not as the main thumbnail
hook.

Concept jobs:

- `skills-are-dependencies`: thesis match.
- `prompt-files-are-code`: contrarian hook.
- `eval-your-skills`: narrower engineering hook around evals.
