## Current recommendation

Use `not-just-prompts` first. It has the clearest hook, the portrait is strong,
and the text matches the opening argument. `claude-code-stack` is a good backup
if you want a more search-led title and thumbnail pair.

## Thumbnail prompt notes

The thumbnail should use Daniel's portrait as the creator anchor and keep the
repo context secondary. The video is about engineering structure: generated
skills, browser automation, evals, memory, and review.

Concept jobs:

- `not-just-prompts`: broad reach and strongest hook.
- `agent-system`: thesis match.
- `claude-code-stack`: source-aware version for people searching Claude Code.
